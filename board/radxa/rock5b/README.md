# HAOS Rock5B 支持

## 项目概述
本项目为Home Assistant OS(HAOS)提供对Radxa Rock5B开发板的支持，包含：
- 定制化的Linux内核和系统镜像
- 硬件适配（RK3588芯片组、GPIO、外设等）
- 自动存储管理（动态分区扩容）
- OTA更新支持
- 优化的系统服务集成

## 硬件要求
- Radxa Rock5B开发板
- 至少16GB eMMC存储或SD卡
- 推荐4GB以上内存

## 安装指南
1. 下载最新HAOS镜像
2. 使用工具(如balenaEtcher)将镜像写入存储设备
3. 插入Rock5B并启动
